# Ijaiye MFM Branch Website

**Live Site:** [ijaiye-church-website.vercel.app](https://ijaiye-church-website.vercel.app)

## 📖 Project Overview

The **Ijaiye Mountain of Fire and Miracles Ministries (MFM)** branch previously had no dedicated website, relying solely on the headquarters' main website for online visibility.  
This created a digital gap — members, visitors, and the local community could not easily access branch-specific information, updates, or events.

This project aims to **solve the digital presence challenge** of the Ijaiye branch by creating a **modern, responsive, and fast-loading website** specifically tailored to our local congregation.

The website acts as a **digital extension of the church**, offering:
- Service schedules and event updates
- Information about the branch leadership
- Resources for spiritual growth
- Contact and location details for new visitors

## 🎯 Purpose

The primary goal is to:
- Establish an **independent online presence** for the Ijaiye branch while maintaining the MFM brand identity.
- Improve accessibility for both local and international members.
- Enhance engagement through an easy-to-navigate platform.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (TypeScript)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Hosting & Deployment:** [Vercel](https://vercel.com/)

## 🚀 Features

- 📱 **Responsive Design** – Optimized for mobile, tablet, and desktop devices.
- ⚡ **Fast Loading Speed** – Built with performance and SEO in mind.
- 🎨 **Clean UI** – A modern, welcoming design aligned with MFM’s visual identity.
- 🗓️ **Events & Service Times** – Updated schedule for members and visitors.
- 📍 **Contact & Location Info** – Easy to find and navigate.
